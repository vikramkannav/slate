
#Categories

##Categories

```shells
curl "http://base_url/api/categories" 
 -H " Authentication : bearer access token"
```

> The above command returns JSON structured like this:

```json
  {
  	"categories": [{
  			"id": 1,
  			"type": "musician"
  		},
  		{
  			"id": 2,
  			"type": "gitarist"
  		}
  	]
  }
```

###HTTP Request

`GET http://base_url.com/api/categories`
